Версия 1.0
Реализовано:
1. Соединие с сервером ("recon" для переподключения)
2. Разрыв с сервером (комманда "quit")
3. Ввод имени
3. Запрос справки (комманда "help", однако команда работает крайне странно,
считывается почему-то только часть справки отправленой с сервера)
И еще замечен баг, если вводить много комманды, то после 3-ей идет крашинг программы(допустим подряд 3 раза отправить коммандду "help"),
по неизвестным причинам.
При запуске программы клиента потребуется ввести имя пользователя,после будет предоставлен доступ к командой строке.
